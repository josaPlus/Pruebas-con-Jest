# Proyecto de Pruebas en Node.js con Jest

Este proyecto contiene una serie de funciones en JavaScript creadas para realizar pruebas unitarias utilizando **Jest**.

El objetivo del proyecto es demostrar diferentes tipos de funciones, manejo de parámetros, retornos y manejo de errores.

---

# Requisitos

Antes de ejecutar el proyecto necesitas tener instalado:

* Node.js (versión 16 o superior recomendada)
* npm (se instala junto con Node)

Puedes verificarlo con:

```
node -v
npm -v
```

---

# Clonar el repositorio

Clona el repositorio en tu computadora:

```
git clone <URL_DEL_REPOSITORIO>
cd <NOMBRE_DEL_REPOSITORIO>
```

---

# Instalar dependencias

Instala todas las dependencias necesarias con:

```
npm install
```

Esto descargará automáticamente todos los paquetes definidos en `package.json`.

---

# Ejecutar las pruebas

Para correr las pruebas unitarias usa:

```
npm test
```

o directamente con:

```
npx jest
```

---
